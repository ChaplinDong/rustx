# 01 入门：从零搭建 Rust 开发环境
## 章节说明
本章节专为 **纯零基础新手** 设计，无需任何编程基础、无需提前安装任何工具，全程手把手实操，帮你快速完成 Rust 入门的核心准备工作。
核心目标：学会搭建 Rust 开发环境、掌握官方工具 Cargo 的基础用法、写出并运行第一个 Rust 程序，为后续学习打下坚实基础。

## 适用人群
- 完全没有接触过 Rust 语言的新手
- 有其他编程语言基础（如 Java、Python、Go 等），想快速上手 Rust 的开发者
- 想了解 Rust 开发流程，准备入门 Rust 的学习者

## 学习前准备
1.  一台正常运行的电脑（Windows 10 及以上、macOS 10.13 及以上、Linux 主流发行版，如 Ubuntu 20.04+）
2.  稳定的网络（用于下载 Rust 工具链，无需科学上网）
3.  耐心和细心（每一步严格按照教程操作，避免遗漏细节）

## 学习内容 & 学习顺序（必严格遵循）
1.  01-install-rust.md：搭建全平台 Rust 开发环境（核心第一步，环境没搭好，后续无法操作）
2.  02-cargo-basics.md：学习 Cargo 核心用法（Rust 官方工具，管理项目、编译运行全靠它）
3.  03-first-program.md：编写并运行第一个 Rust 程序（Hello World，建立实操信心）
4.  04-common-issues.md：排查新手常见问题（遇到报错不用慌，这里有解决方案）

## 注意事项
1.  全程不要跳过任何步骤，尤其是环境搭建部分，每一步都要确认操作成功后再进行下一步
2.  输入命令时，严格按照教程中的写法，不要多输、少输字符（区分大小写，Rust 对大小写敏感）
3.  遇到报错不要慌张，先查看 04-common-issues.md，大部分新手报错都能在这里找到解决方案
4.  本章节所有操作均无需管理员权限（Windows 双击安装程序时，无需选择“以管理员身份运行”）
5.  学习完成后，建议保留本章节文档，后续遇到环境、工具相关问题可随时查阅
