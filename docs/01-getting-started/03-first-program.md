# 03 第一个 Rust 程序：Hello World（详细实操版）
通过一个简单的 Hello World 程序，熟悉 Rust 代码的基本结构、编写规范、编译运行流程，逐行解读代码含义，补充新手可能遇到的语法问题，让你快速入门 Rust 代码编写，建立实操信心。

本教程全程基于上一节创建的 Cargo 项目，无需额外创建项目，直接在已有项目中修改代码即可，新手建议严格按照步骤操作，不要跳过任何细节。

## 一、前提准备
1.  已完成 Rust 环境搭建（01-install-rust.md），Cargo 已安装成功
2.  已创建 Cargo 项目（参考 02-cargo-basics.md），这里以 `hello-rust` 项目为例
3.  已进入项目目录（终端中执行 `cd hello-rust`）

## 二、步骤1：打开代码文件（src/main.rs）
Rust 程序的所有代码都放在 `src` 目录下，`main.rs` 是主程序入口文件，默认已经生成了 Hello World 代码，我们先打开这个文件，查看并修改代码。

### 打开方式（全平台通用，新手推荐）
1.  找到你创建的 `hello-rust` 项目文件夹（如桌面）
2.  双击打开文件夹，找到 `src` 文件夹，双击打开
3.  找到 `main.rs` 文件，用文本编辑器打开（推荐使用 VS Code、Notepad++、Sublime Text 等，也可以用系统自带的记事本，但不推荐，因为没有语法高亮）
    - 若没有安装文本编辑器，Windows 可使用记事本，macOS 可使用文本编辑（需将格式设置为“纯文本”）
4.  打开后，可看到默认生成的代码（如下），我们先解读这段代码，再进行修改：
```rust
fn main() {
    println!("Hello, world!");
}
```

## 三、步骤2：逐行解读代码（新手必懂，建立语法认知）
这段代码虽然简单，但包含了 Rust 程序的核心结构，逐行解读，让你明白每一行的作用，为后续学习打下基础：

### 第1行：fn main() {
- `fn`：是 Rust 中的关键字，用于定义一个函数（函数是 Rust 程序的基本组成单元，用于封装一段可执行代码）
- `main`：函数名，是 Rust 程序的**入口函数**，也是程序的起始点——无论代码有多少行，程序都会从 `main` 函数开始执行
- `()`：括号内用于填写函数的参数，这里 `main` 函数没有参数，所以括号为空
- `{}`：花括号用于包裹函数体，所有属于 `main` 函数的代码，都必须放在这对花括号内，花括号必须成对出现，且左花括号 `{` 要和 `main()` 在同一行（Rust 语法规范）

### 第2行：    println!("Hello, world!");
- 前面的4个空格：是缩进，Rust 推荐使用4个空格作为缩进（也可以用1个制表符，但不推荐，避免格式混乱），缩进的作用是区分代码层级，让代码更易读，新手必须养成缩进的习惯
- `println!`：这是 Rust 中的**宏**（不是函数，后续章节会详细讲解宏），作用是在终端中打印指定的内容，后面的 `!` 是宏的标志，用于区分宏和普通函数
- `""`：双引号内是要打印的字符串（文本内容），可以修改为任意你想打印的内容，如 `Hello, RustX!`、`Rust 入门第一步！`
- `;`：分号是 Rust 语法的重要标志，用于表示一条语句的结束，每一条可执行语句的结尾都必须加 `;`，否则会报错（新手最容易遗漏，一定要记住）

### 第3行：}
- 右花括号，用于结束 `main` 函数，和第1行的左花括号 `{` 对应，必须成对出现，若遗漏右花括号，会导致编译失败

## 四、步骤3：修改代码（实操练习，加深印象）
默认代码打印的是 `Hello, world!`，我们修改为 `Hello, RustX!`（贴合你的 rustx 项目），步骤如下：
1.  在打开的 `main.rs` 文件中，找到第2行的 `Hello, world!`
2.  将其修改为 `Hello, RustX!`，修改后的代码如下：
```rust
fn main() {
    println!("Hello, RustX!");
}
```
3.  保存文件（快捷键：Ctrl + S / Command + S），确保修改生效，不要忘记保存，否则编译的还是原来的代码

## 五、步骤4：运行程序（查看结果）
代码修改完成后，我们通过 Cargo 运行程序，查看修改后的效果，步骤如下：
1.  打开终端，确保当前路径是 `hello-rust` 项目目录（若不在，执行 `cd hello-rust` 进入）
2.  输入命令：`cargo run`，按回车执行
3.  等待编译完成，编译成功后，终端会显示以下内容：
```
   Compiling hello-rust v0.1.0 (/Users/xxx/Desktop/hello-rust)
    Finished dev [unoptimized + debuginfo] target(s) in 0.2s
     Running `target/debug/hello-rust`
Hello, RustX!
```
4.  最后一行 `Hello, RustX!` 就是我们程序的运行结果，说明代码修改成功、程序运行正常！

## 六、步骤5：常见代码错误及解决方法（新手避坑）
新手编写代码时，很容易出现以下错误，提前了解，遇到报错不用慌：

### 错误1：遗漏分号 `;`
#### 错误代码
```rust
fn main() {
    println!("Hello, RustX!") // 遗漏了分号
}
```
#### 报错提示
`expected ';' found '}'`（期望分号，却找到了右花括号）
#### 解决方案
在 `println!` 语句结尾添加分号 `;`，修改后即可正常编译运行。

### 错误2：遗漏右花括号 `}`
#### 错误代码
```rust
fn main() {
    println!("Hello, RustX!");
// 遗漏了右花括号
```
#### 报错提示
`unclosed delimiter: {`（未闭合的分隔符：{）
#### 解决方案
在代码最后添加右花括号 `}`，确保和左花括号 `{` 成对出现。

### 错误3：缩进错误（多缩进/少缩进）
#### 错误代码（少缩进）
```rust
fn main() {
println!("Hello, RustX!"); // 没有缩进
}
```
#### 报错提示
`expected indentation, found literal`（期望缩进，却找到了文本）
#### 解决方案
在 `println!` 语句前面添加4个空格，保持正确的缩进。

### 错误4：宏的 `!` 遗漏
#### 错误代码
```rust
fn main() {
    println("Hello, RustX!"); // 遗漏了 !
}
```
#### 报错提示
`expected function, found macro`（期望函数，却找到了宏）
#### 解决方案
在 `println` 后面添加 `!`，改为 `println!`。

### 错误5：字符串引号不闭合
#### 错误代码
```rust
fn main() {
    println!("Hello, RustX!); // 遗漏了右双引号
}
```
#### 报错提示
`unterminated string literal`（未闭合的字符串）
#### 解决方案
在字符串末尾添加右双引号 `"`，确保双引号成对出现。

## 七、小练习（可选，巩固实操）
完成基础修改后，可尝试以下小练习，加深对代码的理解和实操能力：
1.  修改代码，打印 `Hello, Rust! 我是新手，正在入门 Rust 开发！`
2.  打印多行内容，示例代码如下（可直接复制运行）：
```rust
fn main() {
    println!("Hello, RustX!");
    println!("我正在学习 Rust 开发");
    println!("这是我的第一个 Rust 程序！");
}
```
3.  运行练习代码，查看终端输出结果，确认程序能正常运行。

## 八、核心总结（新手必记）
1.  Rust 程序的入口是 `main` 函数，程序从 `main` 函数开始执行
2.  每条可执行语句的结尾必须加 `;`（分号是语句结束的标志）
3.  代码缩进推荐使用4个空格，养成良好的代码格式习惯
4.  `println!` 是宏，用于打印内容，后面必须加 `!`
5.  字符串必须用双引号 `"` 包裹，且必须成对出现
6.  花括号 `{}` 用于包裹函数体，必须成对出现，不能遗漏

## 九、拓展说明（可选，了解即可）
1.  代码注释：Rust 支持两种注释方式，用于添加代码说明，不影响程序运行：
    - 单行注释：用 `//` 开头，后面的内容为注释，如 `// 这是单行注释`
    - 多行注释：用 `/* */` 包裹，中间的内容为注释，如 `/* 这是多行注释，可换行 */`
    示例：
    ```rust
    // 这是主函数，程序的入口
    fn main() {
        /* 这是打印语句，
        用于打印 Hello, RustX! */
        println!("Hello, RustX!");
    }
    ```
2.  代码格式：Rust 有严格的代码格式规范，建议新手使用 `cargo fmt` 命令，自动格式化代码（进入项目目录，执行 `cargo fmt`，即可自动调整缩进、空格等格式）
